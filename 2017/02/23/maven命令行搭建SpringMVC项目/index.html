Maven命令行搭建SpringMVC项目

发表于 2017-02-23 | 分类于 linux ， maven | |

利用 MAVEN 构建 web 项目骨架

建立包名为 com.app, 项目名为 forfree的普通JavaWeb项目

建立完成后，需要补全maven项目目录结构，添加以下目录：

./src/main/java
./src/test/java
./src/test/resources
./src/test/webapp
./src/test/webapp/WEB-INF

编辑pom.xml文件，引入spring框架及其相应依赖

编辑web.xml文件，配置servlet运行环境

在/WEB-INF目录下建立并编辑spring-servlet.xml文件，配置spring运行状态

建立控制器

根据spring-servlet.xml中context给出的扫描路径，建立相应的包，并在其中建立控制器testController.java

在/WEB-INF/views目录下建立视图文件hello.jsp

至此，一个完整的SpringMVC的可运行环境已经搭建完成

项目打包部署

访问 http://localhost:8080/forfree/mvc/hello 即可看到hello.jsp的内容了
